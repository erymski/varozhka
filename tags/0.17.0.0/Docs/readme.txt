------------
* Varozhka *
------------

This is a framework to work with Netflix prize (http://www.netflixprize.com) 
dataset. The framework indexes the dataset and provides interface to access 
the indexes. A user of the framework should implement his algorithm a stub 
which calculates rating for the given customer/movie pair.


*********** Requirements *******************

* 1GB of RAM. More is better.
* Windows 2000, XP, 2003, Vista (?)
* .NET Framework 2.0

To simplify development I would recommend to use Visual Studio 2005 (any edition). 
If you need íou can get a free copy of Express edition at 
http://msdn.microsoft.com/vstudio/express/.
